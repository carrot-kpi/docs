Welcome to the Carrot documentation portal, your go-to resource for everything
Carrot.

From user to technical documentation, this portal is meant to be a reference
point for anyone interested in interacting or building on top of Carrot, an open
protocol facilitating the implementation of novel incentivization mechanisms
through the **KPI token** primitive.
